How do we generate an access token?

Workflow:
1. Access the login page, enter a username + Password (code challenge)
2. Page will redirect to the url that we provided at setup + attach a special code in the url query parameter
3. Take this code and make another API call to Cognito and exchange the code with the token (code_verifier)
